# Electric Cars Tableau Project

## Project Overview
This project demonstrates how to collect EV datasets, store them in a MySQL database, perform SQL operations, and connect the database to Tableau for visualization.

## Tools Used
- MySQL Workbench
- MySQL Server 8.0
- Tableau Desktop
- SQL

## Dataset
- ElectricCarData_Clean
- EVIndia
- Electric Vehicle Charging Station List
- Cheapest Electric Cars Database

## Steps Completed
- Collected datasets
- Created MySQL database
- Imported CSV files
- Performed SQL operations
- Connected MySQL to Tableau

## Tableau Visualizations

Completed the following Tableau visualizations:

1. Charging Stations by Region and Type
2. EV Charging Stations Map of India
3. Different EV Cars in India
4. Top Speed for Different Brands
5. Price Comparison of EV Cars
6. Top 10 Most Efficient EV Brands
7. Brands According to Body Style
8. Brand Filtered by PowerTrain

Tools Used:
- Tableau Desktop Professional
- MySQL
- SQL
