# Electric Cars Tableau Project

## Project Overview
This project demonstrates how to collect EV datasets, store them in a MySQL database, perform SQL operations, and connect the database to Tableau for visualization.

## Tools Used
- MySQL Workbench
- MySQL Server 8.0
- Tableau Desktop
- SQL

## Dataset
- ElectricCarData_Clean
- EVIndia
- Electric Vehicle Charging Station List
- Cheapest Electric Cars Database

## Steps Completed
- Collected datasets
- Created MySQL database
- Imported CSV files
- Performed SQL operations
- Connected MySQL to Tableau
