# README.md Contents

## Project Overview

This project demonstrates a complete Business Intelligence workflow using SQL, Tableau Desktop Professional, Tableau Public, and Flask. Multiple Electric Vehicle datasets were integrated into MySQL, analyzed using SQL, visualized through Tableau dashboards and stories, published to Tableau Public, and embedded into a responsive Flask web application.

---

## Project Workflow

1. Collected EV datasets.
2. Imported datasets into MySQL.
3. Performed SQL data cleaning and analysis.
4. Connected MySQL with Tableau.
5. Built 10 Tableau Worksheets.
6. Designed Interactive Dashboard.
7. Created 5 Story Scenes.
8. Published Dashboard & Story to Tableau Public.
9. Developed a responsive Flask UI.
10. Embedded Tableau Dashboard and Story inside the web application.
